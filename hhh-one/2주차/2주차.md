### 상하좌우

#### [코드]

```python
N = int(input())
move = input().split()
x, y = 1, 1

for i in move:
    if(i == "R" and y <= N):
        y += 1
    elif(i == "L" and y != 1):
        y -= 1
    elif(i == "U" and x != 1):
        x -= 1
    elif(i == "D" and x <= N):
        x += 1
print(x, y)
```

<br>

#### [해결방안]

- 더해지는 R이나 D는 최대 공간을 넘으면 안되고, 빼지는 L이나 U는 1보다 작으면 안되기 때문에 if문으로 같이 제약사항을 넣었다.
- move를 하나씩 비교해가며 이동하였다.

### 시각

#### [코드]

```python
N = int(input())
answer = 0

for h in range(N + 1):
    for m in range(60):
        for s in range(60):
            if '3' in str(h) + str(m) + str(s):
                answer += 1
print(answer)
```

<br>

#### [해결방안]

- 책 코드를 읽고 풀었다.
- 3중 반복문을 이용해 시, 분, 초에서 3이 들어갈 때 조건으로 카운트를 셌다.

<br>

#### [어려웠던 부분]

- 3이 들어가는 초마다 시,분을 세볼까 했는데 나중에 분이나 시에서 3을 셀 때 겹치는 부분을 빼주어야 하는게 복잡할거라 생각해서 생각을 좀 많이 한 것 같다.
- 3이 들어있을 때 카운트를 세는것을 어떤 방식으로 해야될지 몰랐었다.
