### 상하좌우

#### [코드]

```python
N = int(input())
move = input().split()
x, y = 1, 1

for i in move:
    if(i == "R" and y <= N):
        y += 1
    elif(i == "L" and y != 1):
        y -= 1
    elif(i == "U" and x != 1):
        x -= 1
    elif(i == "D" and x <= N):
        x += 1
print(x, y)
```

<br>

#### [해결방안]

- 더해지는 R이나 D는 최대 공간을 넘으면 안되고, 빼지는 L이나 U는 1보다 작으면 안되기 때문에 if문으로 같이 제약사항을 넣었다.
- move를 하나씩 비교해가며 이동하였다.
